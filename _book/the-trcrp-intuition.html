3 The TRCRP Intuition

TRCRP is a generative temporal extension of the Dirichlet process. Samples from its posterior are distributed according to a Normal distribution, with and Normal-InverseGamma prior. In a multivariate setting it is capable of clustering data according to the states which they aquire via the embedded Chinese Restaurant Process(CRP). It uses a Markov Chain Monte Carlo processes for sampling, and hyper parameter resampling.

BNP based methods such as TRCRP are often used to model data where the dimension of interest may increase with the size of the data set. The Dirichlet Process Mixture Models(DPMM) underlying many BNP models allow for an infinite number of possible categories, or mixture modes. As such it is a possible candidate for the modeling of signals from devices with an unknown, possibly infinite, number of failure modes. Further to this the state attribution, or mode association, these methods utilize is sensitive to the number of datapoints already existing in a given state or mode; unknown or unfamiliar incoming data will tend to not be grouped with familiar data. Again this makes such methods possible candidates for identifying anomalous data points with features that stand out from their peers.
